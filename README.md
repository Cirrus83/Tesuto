# Tesuto
A mini command line toolbox.
## How to use
Use "help" to get more information.
## Available Function(s)
- `help` – Show help information.
- `hello` – Print "Hello World!" on the screen.
- `version` – Show version information.
- `gcd` – Calculate GCD(multiple positive integers supported).
- `inverse` – Calculate inverse matrix automatically.
- `manual` – Calculate inverse matrix by yourself.
  - Available commands: add, sub(subtract), mul(multiply), div(divide), neg(negative).
- `kurikaesu` – 繰り返す. If an integer argument is added, it can print a string for the same times.Else, it prints a string for random times(<2220).
- `ipv6prob` – Compress an IPV6 location(multiple inputs supported with an argument provided).
- `gparescuer` – An interesting game that can help you recover from a test fail.
